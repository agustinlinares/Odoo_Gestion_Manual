# Documentación Odoo 18 (GitHub Pages)

Este repositorio está preparado para publicarse con **GitHub Pages**.

## Publicación
1. Crea un repositorio en GitHub (p. ej. `odoo18-docs`).
2. Sube el contenido de esta carpeta a la rama `main`.
3. En GitHub: **Settings → Pages → Build and deployment**
   - Source: `Deploy from a branch`
   - Branch: `main` / `/ (root)`
4. Guarda y espera a que se publique.

## Estructura
- `index.html`: portada y navegación
- `instalacion.html`: manual de instalación + vídeo + capturas
- `indexStyles.css`: plantilla visual (Ubuntu/Odoo)
- `img/`: Figuras 01–20
